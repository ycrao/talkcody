## 欢迎使用 TalkCody

TalkCody 是一款专为 macOS 开发者设计的强大桌面 AI 编程助手。它采用最前沿的技术构建,包括 React 19、Tauri 2 和 Vercel AI SDK,为您提供直观的 AI 驱动编程辅助界面。

## 核心特性

### 🚀 原生性能

TalkCody 基于 Tauri 2 构建,这意味着您可以获得原生 macOS 性能,而无需承受传统 Electron 应用的臃肿。该应用轻量、快速,完美融入您的 Mac 系统。

### 🎨 精美界面

采用 Tailwind CSS 和 Shadcn UI 组件设计,TalkCody 提供了现代、简洁的界面,支持亮色和暗色模式。界面完全响应式,可自适应您的工作流程。

### 🤖 AI 智能辅助

由 Vercel AI SDK 5.0 驱动,TalkCody 连接到最先进的语言模型,为您提供智能代码建议、解释和重构辅助。

## 安装

开始使用 TalkCody 非常简单:

```bash
# 克隆仓库
git clone https://github.com/talkcody/talkcody.git

# 安装依赖
cd talkcody
bun install

# 运行应用
bun run tauri dev
```

## 您的第一次对话

TalkCody 运行后,您就可以开始与 AI 助手对话了。只需输入您的问题或描述您想要构建的内容,TalkCody 就会帮助您编写、理解或调试代码。

### 对话示例

**代码生成:**
```
用户: 创建一个用户资料卡片的 React 组件
TalkCody: 我将帮您创建一个用户资料卡片组件...
```

**代码解释:**
```
用户: 这个 TypeScript 类型守卫是做什么的?
TalkCody: 这个类型守卫用于检查是否...
```

**调试:**
```
用户: 为什么我的 useState hook 没有更新?
TalkCody: 让我帮您调试一下...
```
