# Editor Settings

Configure TalkCody's integrated Monaco code editor to match your preferences and workflow.

## Overview

The editor settings control:
- Appearance and theme
- Behavior and editing features
- AI completions integration
- File handling
- Keyboard shortcuts

## Appearance

### Theme

**Editor Theme Options**:
- **VS Dark** (default): Classic dark theme
- **VS Light**: Light theme for bright environments
- **High Contrast Dark**: Enhanced contrast for accessibility
- **High Contrast Light**: Light high contrast
- **Custom**: Import VS Code themes

**Setting theme**:
```
Settings → Editor → Theme
Select from dropdown
```

**Sync with System**:
```
☑ Follow system theme
```
Automatically switches between light/dark based on OS settings.

### Font Configuration

**Font Family**
```

```
Default: normal

## Troubleshooting

### Performance Issues

**Solutions**:
- Disable syntax highlighting for large files
- Turn off minimap
- Reduce font size
- Disable AI completions in large files
- Close unused files

### Font Not Appearing

**Check**:
- Font is installed on system
- Font name is correct (case-sensitive)
- Font supports ligatures (if enabled)
- Restart TalkCody after installing fonts

### Shortcuts Not Working

**Debug**:
- Check for conflicts with system shortcuts
- Verify keyboard layout
- Reset to defaults
- Import fresh keybindings

### Auto-Save Not Working

**Verify**:
- Auto-save is enabled
- File permissions allow writes
- File isn't open in another app
- Sufficient disk space
