## Installation

<Steps>

<Step>
### Download and Install TalkCody

Download the latest version of TalkCody for your platform from the [Client Downloads](/docs/introduction/client-downloads) page.

</Step>

<Step>
### Configure API Keys

Before using TalkCody, you need to configure at least one AI provider:

1. Open TalkCody Settings page
2. Click the **API Keys** tab
3. Add an API key for your preferred provider

![API Keys Configuration](https://cdn.talkcody.com/images/api-keys.png)

For details, refer to the [Configure API Keys](/docs/configuration/api-keys) guide.
</Step>


<Step>
### Configure Model

TalkCody currently uses Claude Haiku as the default model. You can change the model as needed:

1. Open TalkCody Settings page
2. Click the **Model** tab
3. Select your preferred model

![AI Model Settings](https://cdn.talkcody.com/images/model-setting.png)

</Step>

<Step>
### Import Project
Import your local code repository.

![Import Projects](https://cdn.talkcody.com/images/import-projects.png)

</Step>

<Step>
### Start AI Coding

Now you're ready to start AI coding with TalkCody! You can ask TalkCody to help you:

- Write code
- Review code
- Generate tests
- Explain concepts
- Answer any questions about your codebase

</Step>

</Steps>
