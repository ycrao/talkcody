## 什么是 Skill？

![TalkCody Skills](https://cdn.talkcody.com/images/skills.png)

Skill 包含：
- **系统提示片段**：注入到智能体的行为指令
- **工作流规则**：特定任务的执行步骤
- **参考文档**：领域知识和最佳实践
- **可执行脚本**：自动化常见任务

通过激活不同的 Skills，智能体可以快速获得特定领域的专业能力。

## 使用场景

- **数据库开发**：SQL 语法、查询优化、数据库设计
- **代码审查**：代码规范、安全检查、性能分析
- **文档编写**：API 文档、技术规格、用户指南
- **测试开发**：单元测试、集成测试、测试策略

## 如何使用

### 激活 Skill

1. 在聊天界面点击 **Skills 图标**（拼图图标）
2. 从列表中选择需要的 Skill
3. 选中的 Skill 自动应用到当前对话

### 下载 Skill
1. 进入 **Skills 页面**
2. 查看 "Remote Skills" 标签
3. 点击 **Install** 下载所需的 Skill

### 管理 Skills

**查看已安装 Skills**
- 进入 **Skills 页面**
- 查看 "Local Skills" 标签

**创建自定义 Skill**
1. 点击 **Create New Skill**
2. 填写名称和描述
3. 编写系统提示片段
4. 添加工作流规则（可选）
5. 保存

## 与智能体的关系

Skills 和 Agents 的区别：

| 特性 | Skills | Agents |
|------|--------|--------|
| 作用 | 知识增强 | 完整助手 |
| 配置 | 提示片段 | 完整系统提示 |
| 切换 | 可同时激活多个 | 一次使用一个 |
| 适用 | 特定领域知识 | 特定任务流程 |

<Callout type="tip">
一个 Agent 可以同时使用多个 Skills，获得多领域的专业能力。
</Callout>

## 最佳实践

### 编写 Skill 提示

- 聚焦单一领域
- 提供具体指导
- 包含示例输出格式
- 避免与其他 Skills 冲突
