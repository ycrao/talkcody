TalkCody 的所有重要更新都记录在这里。

## [0.2.2] - 2025-12-30

### 新增

- **页面统一**：将 Explore 页面和 Chat 页面统一，提供更连贯的用户体验

### 优化

- **优化上下文压缩策略**：改进消息过滤和压缩逻辑，更好地支持长任务（Long Run Task）
- **优化 GPT-5.2 Codex 体验**：优化通过 OpenAI 订阅使用的提示词
- **优化子 Agent 调用**：改进了调用子 Agent 时的超时处理机制

### 修复

- **修复 GitHub Copilot 图片处理**：解决了 GitHub Copilot provider 处理图片请求时的 Bug
- **修复编辑器同步**：修复了编辑器无法及时感知外部文件修改的问题
- **修复 UI 显示**：修复了 Edit File 结果 UI 在处理大文件时 Diff 显示不对的 Bug

## [0.2.1] - 2025-12-27

### 新增

- 新增 Qwen Code 免费提供商，节省您的编码成本
- 新增 Github Copilot 免费提供商，节省您的编码成本

### 优化

- 优化上下文压缩逻辑，更好地处理长对话

## [0.2.0] - 2025-12-25

### 新增

- **支持 通过 Claude Pro/Max 订阅，在 TalkCody 中使用 Claude 模型**
- **通过 ChatGPT Plus/Pro 订阅，在 TalkCody 中使用 OpenAI 模型**
- **LSP 支持**：完整实现语言服务器协议，提供代码导航、代码补全、跳转到定义、查找引用等核心功能，大幅提升代码理解和编辑体验
- **Exa 免费网页搜索**：新增 Exa 搜索引擎提供商，让网页搜索开箱即用
- **终端字体自定义**：支持自定义终端字体和字号设置
- 支持 MiniMax 2.1 模型 （**可以通过 TalkCody Provider 免费使用**）
- 支持 GLM 4.7 模型 （TalkCody 支持模型热更新，一般最新的优秀 Coding 模型12小时会自动更新）

### 优化

- 优化 write file 和 edit file tool 的结果展示
- 优化 Prompt Cache，进一步节省成本

### 修复

- 修复 Windows 终端相关 bug
- 修复模型选择器列表重复显示 bug
- 修复 write file tool 在某些情况下无法正确写入文件的 bug
- 修复多窗口下，git status 显示不对的 bug 

## [0.1.20] - 2025-12-19

### 新增

- MiniMax M2.1 内测模型支持，所有用户均可免费体验。模型选择 MiniMax M2.1 Beta，提供商选择 talkcody 即可使用。

### 优化

- 优化 Plan Mode 审批和 Diff Review UI

## [0.1.19] - 2025-12-18

### 新增

- 多 Agent 并行执行（实验版本），支持多个 Agent 同时执行任务
- 基于 Git Worktree 的 Task 并行执行（实验版本），支持多个 Task 在独立工作目录中并行运行
- 一键 Git Commit：在文件变更摘要中新增 Commit 按钮，AI 自动生成提交信息
- 一键 Code Review：在文件变更摘要中新增 Review 按钮，一键调用 Code Review Agent

### 优化

- 优化 MCP 工具选择按钮
- 优化本地 Agent 加载性能
- 改进 Edit File 工具
- 改进 Bash 工具的危险命令检测
- 优化 Context Compaction 逻辑
- 优化 AI 请求的重试策略

### 修复

- 修复 Windows 终端的 bug
- 修复 全局内容搜索空格直接退出的 bug

## [0.1.18] - 2025-12-13

### 新增

- 编辑器代码检查功能 (Editor Lint)，支持实时语法检查和快速修复
- GPT 5.2 模型支持
- 新增功能文档：Coding Plan、网页搜索、语音输入、代码检查

### 优化

- 改进网页搜索工具
- 改进 Web fetch 工具

### 修复

- 修复 Windows PowerShell 终端 bug
- 修复没有正确获取 bashrc/zshrc 导致环境变量缺失的问题

## [0.1.17] - 2025-12-11

### 新增

- 支持多个会话并行执行, 大幅提升工作流效率
- 支持自定义 AI 提供商
- 新增内置 AI 提供商：moonshot
- 新增内置模型：GLM-4.6 v
- 新增内置 Minimax Coding Plan MCP，支持网页搜索和图像输入，拥有 Minimax Coding Plan 的用户可以直接使用
- 新增内置 GLM Coding Plan MCP，支持网页搜索和图像输入，拥有 GLM Coding Plan 的用户可以直接使用
- 版本更新后，向用户展示更新日志

### 优化

- 优化 Bash 工具输出以提升性能
- 优化 Github PR 工具输出以提升性能
- 优化 Context Compaction 逻辑，提升多轮对话性能

### 修复

- 修复 HTTP MCP 服务器请求头配置问题
- 修复 Stdio MCP 服务器不支持自定义环境变量的问题
- 修复 多窗口时，数据库提前退出时的问题
