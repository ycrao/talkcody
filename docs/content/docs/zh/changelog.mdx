TalkCody 的所有重要更新都记录在这里。

## [0.1.20] - 2025-12-19

### 新增

- MiniMax M2.1 内测模型支持，所有用户均可免费体验。模型选择 MiniMax M2.1 Beta，提供商选择 talkcody 即可使用。

### 优化

- 优化 Plan Mode 审批和 Diff Review UI

## [0.1.19] - 2025-12-18

### 新增

- 多 Agent 并行执行（实验版本），支持多个 Agent 同时执行任务
- 基于 Git Worktree 的 Task 并行执行（实验版本），支持多个 Task 在独立工作目录中并行运行
- 一键 Git Commit：在文件变更摘要中新增 Commit 按钮，AI 自动生成提交信息
- 一键 Code Review：在文件变更摘要中新增 Review 按钮，一键调用 Code Review Agent

### 优化

- 优化 MCP 工具选择按钮
- 优化本地 Agent 加载性能
- 改进 Edit File 工具
- 改进 Bash 工具的危险命令检测
- 优化 Context Compaction 逻辑
- 优化 AI 请求的重试策略

### 修复

- 修复 Windows 终端的 bug
- 修复 全局内容搜索空格直接退出的 bug

## [0.1.18] - 2025-12-13

### 新增

- 编辑器代码检查功能 (Editor Lint)，支持实时语法检查和快速修复
- GPT 5.2 模型支持
- 新增功能文档：Coding Plan、网页搜索、语音输入、代码检查

### 优化

- 改进网页搜索工具
- 改进 Web fetch 工具

### 修复

- 修复 Windows PowerShell 终端 bug
- 修复没有正确获取 bashrc/zshrc 导致环境变量缺失的问题

## [0.1.17] - 2025-12-11

### 新增

- 支持多个会话并行执行, 大幅提升工作流效率
- 支持自定义 AI 提供商
- 新增内置 AI 提供商：moonshot
- 新增内置模型：GLM-4.6 v
- 新增内置 Minimax Coding Plan MCP，支持网页搜索和图像输入，拥有 Minimax Coding Plan 的用户可以直接使用
- 新增内置 GLM Coding Plan MCP，支持网页搜索和图像输入，拥有 GLM Coding Plan 的用户可以直接使用
- 版本更新后，向用户展示更新日志

### 优化

- 优化 Bash 工具输出以提升性能
- 优化 Github PR 工具输出以提升性能
- 优化 Context Compaction 逻辑，提升多轮对话性能

### 修复

- 修复 HTTP MCP 服务器请求头配置问题
- 修复 Stdio MCP 服务器不支持自定义环境变量的问题
- 修复 多窗口时，数据库提前退出时的问题
