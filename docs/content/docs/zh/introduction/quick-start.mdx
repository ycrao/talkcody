## 安装

<Steps>

<Step>
### 下载安装 TalkCody

从[客户端下载](/zh/docs/introduction/client-downloads)页面下载适用于您平台的最新版本 TalkCody。

</Step>

<Step>
### 配置 API 密钥

在使用 TalkCody 之前，您需要配置至少一个 AI 提供商：

1. 打开 TalkCody 设置页面
2. 点击 **API 密钥** 标签
3. 为您首选的提供商添加 API 密钥

![API 密钥配置](https://cdn.talkcody.com/images/api-keys.png)

具体可以参考 [配置 API 密钥](/zh/docs/configuration/api-keys) 指南。
</Step>


<Step>
### 配置模型

TalkCody 目前默认使用 Claude Haiku 模型，您可以根据需要更改模型：

1. 打开 TalkCody 设置页面
2. 点击 **模型** 标签
3. 选择您首选的模型

![AI 模型设置](https://cdn.talkcody.com/images/model-setting.png)

</Step>

<Step>
### 导入项目
导入你的本地代码仓库。

![导入项目](https://cdn.talkcody.com/images/import-projects.png)

</Step>

<Step>
### 开始 AI Coding

现在您已经准备好开始使用 TalkCody 进行 AI 编码了！你可以让 TalkCody 帮你

- 写代码
- 审查代码
- 生成测试
- 解释概念
- 回答关于代码仓库的任何问题

</Step>

</Steps>
